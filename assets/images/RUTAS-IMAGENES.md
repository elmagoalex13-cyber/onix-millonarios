# Rutas de imagenes

Guarda cada imagen con el nombre indicado, usando cualquiera de estas extensiones:

`webp`, `avif`, `png`, `jpg`, `jpeg`, `jfif`, `gif`, `svg` o `bmp`.

Ejemplo: `assets/images/hero/hero-escritorio.jpg`
